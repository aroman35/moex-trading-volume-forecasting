# MOEX Trading Volume Forecasting

Проект по моделированию и прогнозированию ежемесячного объёма торгов Московской биржи с макроэкономическими предикторами, анализом стационарности, ансамблевыми моделями и анализом структурных сдвигов.

## Quick Start

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt

python main.py all
python main.py best-model
python main.py structural
```

## Обзор проекта

- скачивает и агрегирует месячные данные по IMOEX и макроэкономическим факторам;
- строит графики временного ряда и STL-декомпозицию;
- проводит тесты стационарности;
- обучает набор моделей прогнозирования;
- анализирует лучшую ансамблевую модель;
- выявляет структурные сдвиги и аномалии.

## Формулы расчёта производных признаков

Некоторые признаки не загружаются напрямую из внешнего источника, а рассчитываются в коде на основе дневных данных.

### Реализованная волатильность $RV_t$

Рассчитывается как сумма квадратов дневных лог-доходностей индекса IMOEX внутри месяца:

$$
RV_t = \sum_{d \in t} r_d^2
$$

где

$$
r_d = \ln\left(\frac{P_d}{P_{d-1}}\right)
$$

Здесь:

- $P_d$ — дневное значение `CLOSE` индекса IMOEX;
- $P_{d-1}$ — значение `CLOSE` в предыдущий торговый день;
- $r_d$ — дневная лог-доходность;
- $RV_t$ — реализованная волатильность в месяце $t$.

### Количество торговых дней $trading\_days_t$

Рассчитывается как число доступных торговых дат в месяце:

$$ 
trading\_days_t = \sum_{d \in t} 1
$$

где суммирование идёт по всем торговым дням месяца `t`.

Здесь:

- $trading\_days_t$ — количество торговых дней в месяце $t$;
- $d \in t$ — торговые дни, принадлежащие месяцу $t$.

### Логарифм объёма торгов $log\_volume_t$

Используется в задачах визуального анализа, тестах стационарности и моделировании:

$$
log\_volume_t = \ln(V_t)
$$

Здесь:

- $V_t$ — месячный объём торгов;
- $log\_volume_t$ — логарифм месячного объёма торгов.

### Первая разность логарифма

Применяется для анализа стационарности:

$$
\Delta log\_volume_t = log\_volume_t - log\_volume_{t-1}
$$

Здесь:

- $\Delta log\_volume_t$ — первая разность логарифма объёма торгов;
- $log\_volume_{t-1}$ — значение логарифма объёма в предыдущем месяце.

### Сезонная разность логарифма

Используется для анализа сезонной компоненты с годовым лагом:

$$
\Delta_{12} log\_volume_t = log\_volume_t - log\_volume_{t-12}
$$

Здесь:

- $\Delta_{12} log\_volume_t$ — сезонная разность с лагом 12;
- $log\_volume_{t-12}$ — значение логарифма объёма торгов в том же месяце предыдущего года.
